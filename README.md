# Employee Salary Slip Generator

## Description  
This project is a simple C program that takes employee details such as ID, name, designation, and basic salary. It calculates HRA, DA, PF, gross salary, and net salary, and generates an employee salary slip in a text file.

## Features  
- Takes employee details as input  
- Calculates salary components (HRA, DA, PF)  
- Calculates gross and net salary  
- Generates a salary slip in a text file  

## Technologies Used  
- Programming Language: C  
- Concepts Used: Structures and File Handling  

## How to Run  
1. Compile the program  
   gcc salary.c -o salary  

2. Run the program  
   ./salary  

3. Enter the employee details when prompted  
4. Check the generated `salary_slip.txt` file  

## Output  
- A text file named `salary_slip.txt` containing the employee salary details
